# Projectname Cpu

## 1 Introduction
This Document focuses on the CPU/Instruction set Architecture of the CPU for the Projectname.

### 1.1 Design goals
The CPU has 2 main design goals.

1. *The CPU should be fast*
2. *The CPU should be a RISC CPU*
